1Ô∏è‚É£ Installing Python and Jupyter

Why Use Python and Jupyter?

‚úî Python is a powerful language for data analysis, statistics, and visualization.
‚úî Jupyter makes working with Python easier by providing an interactive notebook interface.

Step 1: Install Python

  1. Go to Python.org
  2. Download and install Python for your operating system

  1. Go to Anaconda
  2. Download Anaconda Distribution and install it

Step 3: Open Jupyter Notebook

When you open Jupyter Notebook, you'll see:
üìå Code Cells ‚Äì Write and run Python code interactively
üìå Markdown Cells ‚Äì Write and format documentation
üìå Output Area ‚Äì View results of your code
üìå File Browser ‚Äì Navigate and manage your files

2Ô∏è‚É£ Basic Python Syntax

Step 1: Assigning Variables

cases = 150 # Number of malaria cases
fatality_rate = 4.2 # Fatality rate in percentage
region = "West" # Region name (text)
outbreak = True # Logical value (Boolean)

üîç Explanation:

‚úî cases = 150 assigns 150 to cases
‚úî fatality_rate = 4.2 stores 4.2 in fatality_rate
‚úî region = "West" stores "West" as a text string
‚úî outbreak = True stores a Boolean value (True/False)

Step 2: Checking Data Types

type(cases) # Output: <class 'int'>
type(region) # Output: <class 'str'>
type(outbreak) # Output: <class 'bool'>

üîç Explanation:

‚úî type() function shows the data type of each variable
‚úî 'int' and 'float' mean numbers
‚úî 'str' means text
‚úî 'bool' means True or False

Step 3: Basic Operators

total_cases = cases + 50 # Adding 50 more cases
fatalities = cases * (fatality_rate / 100) # Calculating fatalities
recovered = cases - fatalities # Subtracting fatalities from total cases

üîç Explanation:

‚úî cases + 50 adds 50 cases
‚úî cases * (fatality_rate / 100) converts percentage to decimal and calculates fatalities
‚úî cases - fatalities finds the number of recovered patients

3Ô∏è‚É£ Jupyter Notebooks for Documentation

What are Jupyter Notebooks?

üìå Jupyter Notebooks let you create reports, notebooks, and presentations with text, Python code, and results in one document.

üîç Explanation:

üìå Click Run in Jupyter Notebook to execute a code cell.

4Ô∏è‚É£ Reading Malaria Datasets (CSV, Excel) in Python

Why Read External Data?

üìå Malaria data is stored in CSV (Comma-Separated Values) or Excel files.
üìå We use pandas to load these files into Python.

Step 1: Install Required Packages

pip install pandas # Install pandas package for data analysis
pip install openpyxl # Install openpyxl package for Excel files

üîç Explanation:

‚úî pip install pandas installs pandas (for data analysis)
‚úî pip install openpyxl installs openpyxl (for Excel support)

üìÇ 4.1 Reading CSV Files with pandas

import pandas as pd # Load pandas package

# Read malaria data from a CSV file
malaria_data = pd.read_csv("malaria_cases.csv")

# View first few rows
malaria_data.head()

üîç Explanation:

‚úî import pandas as pd loads the pandas package with alias pd
‚úî pd.read_csv("malaria_cases.csv") reads the CSV file into a DataFrame
‚úî malaria_data stores the imported dataset
‚úî malaria_data.head() shows the first 5 rows of the dataset

üìÇ 4.2 Reading Excel Files with pandas

import pandas as pd # Load pandas package

# Read malaria data from an Excel file
malaria_data_excel = pd.read_excel("malaria_cases.xlsx", sheet_name=0)

# View first few rows
malaria_data_excel.head()

üîç Explanation:

‚úî pd.read_excel("malaria_cases.xlsx", sheet_name=0) reads the first sheet of the Excel file
‚úî malaria_data_excel stores the imported dataset
‚úî malaria_data_excel.head() shows the first 5 rows

üìå 4.3 Writing Data to CSV and Excel

Saving as CSV

malaria_data.to_csv("cleaned_malaria_data.csv", index=False)

üîç Explanation:

‚úî malaria_data.to_csv("cleaned_malaria_data.csv", index=False) saves malaria_data as a CSV file without the index column

Saving as Excel

malaria_data.to_excel("cleaned_malaria_data.xlsx", index=False)

üîç Explanation:

‚úî malaria_data.to_excel("cleaned_malaria_data.xlsx", index=False) saves data as an Excel file without the index column

4Ô∏è‚É£ Bonus Challenge: Automate CSV Import

# Importing pandas
import pandas as pd

# Asking the user for the file path
file_path = input("Enter the path to the CSV file: ")

# Reading the dataset from the provided path
try:
malaria_data = pd.read_csv(file_path)

# Displaying summary statistics
print("Summary Statistics:")
print(malaria_data.describe())

# Saving the cleaned data to a new CSV file
clean_file_path = "cleaned_" + file_path.split("/")[-1]
malaria_data.to_csv(clean_file_path, index=False)
print(f"Cleaned data saved to {clean_file_path}")
except FileNotFoundError:
print(f"Error: The file {file_path} was not found.")
except Exception as e:
print(f"An error occurred: {e}")

Explanation:

- input() is used to ask the user for input (file path).
- describe() gives the summary statistics of the dataset.
- to_csv() saves the DataFrame as a CSV file.
- We use try/except blocks for error handling.
