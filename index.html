2Ô∏è‚É£ Basic Python Syntax

Step 1: Assigning Variables

üîç Explanation:

‚úî fatality_rate = 4.2 stores 4.2 in fatality_rate
‚úî region = "West" stores "West" as a text string
‚úî outbreak = True stores a Boolean value (True/False)

Step 2: Checking Data Types

type(cases) # Output:
type(region) # Output:
type(outbreak) # Output:

üîç Explanation:

‚úî type() function shows the data type of each variable
‚úî 'int' means integer numbers
‚úî 'str' means text strings
‚úî 'bool' means True or False

Step 3: Basic Operators

total_cases = cases + 50 # Adding 50 more cases
fatalities = cases * (fatality_rate / 100) # Calculating fatalities
recovered = cases - fatalities # Subtracting fatalities from total cases

üîç Explanation:

‚úî cases + 50 adds 50 cases
‚úî cases * (fatality_rate / 100) converts percentage to decimal and calculates fatalities
‚úî cases - fatalities finds the number of recovered patients

üìå We use pandas to load these files into Python.

Step 1: Install Required Package

üìÇ 4.1 Reading CSV Files with pandas

# Read malaria data from a CSV file

# View first few rows
malaria_data.head()

üîç Explanation:

‚úî import pandas as pd imports the pandas library with alias 'pd'
‚úî pd.read_csv("malaria_cases.csv") reads the CSV file into a DataFrame
‚úî malaria_data stores the imported dataset
‚úî malaria_data.head() shows the first 5 rows of the dataset

üìÇ 4.2 Reading Excel Files with pandas

malaria_data_excel.head()

üîç Explanation:

‚úî pd.read_excel("malaria_cases.xlsx", sheet_name=0) reads the first sheet (index 0)
‚úî malaria_data_excel stores the imported dataset
‚úî malaria_data_excel.head() shows the first 5 rows

4Ô∏è‚É£ Bonus Challenge: Analyze Malaria Data

# Import pandas import pandas as pd # Read the CSV file malaria_df = pd.read_csv("malaria_data.csv") # Display basic info about the dataset print("Dataset information:") print(f"Number of rows: {len(malaria_df)}") print(f"Columns: {', '.join(malaria_df.columns)}") print("\n") # Calculate summary statistics for cases print("Summary statistics for malaria cases:") print(f"Mean cases: {malaria_df['Cases'].mean():.1f}") print(f"Median cases: {malaria_df['Cases'].median():.1f}") print(f"Maximum cases: {malaria_df['Cases'].max()}") print(f"Total cases across all regions: {malaria_df['Cases'].sum()}") print("\n") # Create a new column for recovery rate malaria_df['Recovery_Rate'] = (malaria_df['Recovered'] / malaria_df['Cases']) * 100 # Display the updated DataFrame print("DataFrame with recovery rate:") print(malaria_df) print("\n") # Save the enriched dataset malaria_df.to_csv("enriched_malaria_data.csv", index=False) print("Enriched dataset saved as 'enriched_malaria_data.csv'")

Explanation:

- We first read the CSV file into a DataFrame.
- We calculate and display basic statistics about the case numbers.
- We create a new column 'Recovery_Rate' that shows what percentage of cases recovered.
- Finally, we save the enriched dataset with the new column to a CSV file.
